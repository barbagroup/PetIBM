Detailed Description

Implementation of parallel decoupled IBPM solver (Li et. al. 2016).

This is an example of using PetIBM to build a parallel incompressible flow solver with the immersed-boundary method proposed by Li et. al. 2016. We name this solver decoupled IBPM".

If readers are interested in using this solver instead of coding, please refer to Running PetIBM, 2D Examples, and 3D Examples.

Reference:

  • Li, R. Y., Xie, C. M., Huang, W. X., & Xu, C. X. (2016). An efficient immersed boundary projection method for flow over complex/moving boundaries. Computers & Fluids, 140, 122-135.
